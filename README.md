# Weather Web App

A simple Weather Web App built using HTML, CSS, and JavaScript. This project allows users to search for a city and view real-time weather information using the OpenWeather API.

## Features

- Search weather by city name
- Real-time temperature and weather details
- Responsive and clean UI
- API integration using OpenWeather

## Technologies Used

- HTML
- CSS
- JavaScript
- OpenWeather API
